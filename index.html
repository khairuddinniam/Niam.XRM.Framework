Niam.XRM.Framework

Fast, easy and lightweight Microsoft Dynamics CRM Plugin Framework that support premise and online edition. One framework to rule them all.

Supported Version

  • Microsoft Dynamic CRM 2015
  • Microsoft Dynamic CRM 2016
  • Microsoft Dynamic CRM 365

Installation

Using nuget package manager:

PM> Install-Package Niam.XRM.Framework

Please see the Niam.Xrm.Framework nuget packages to see the version of supported Microsoft.Crm.Sdk.CoreAssemblies. This framework version number will follow the version number of Microsoft.Crm.Sdk.CoreAssemblies.
